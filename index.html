📘 8051 TRẮC NGHIỆM

🧮 HƯỚNG DẪN FX-580VN X

📌 1. Cộng số HEX (VD: 7A + 3F)
  • MODE → 4 (BASE-N) → màn hình hiện Dec
  • ▶️ (hoặc SHIFT + ▶️) → chọn Hex → màn hình hiện Hex
  • Nhập 7A → + → 3F → =
  • Kết quả: B9 → Đáp án B9H
📌 2. Đổi DEC → BIN (VD: 227 → binary)
  • MODE → 4 (BASE-N) → màn hình hiện Dec
  • Nhập 227 → =
  • ▶️ → chọn Bin → hiển thị 11100011
📌 3. Đổi BIN → DEC (VD: 11100011 → decimal)
  • MODE → 4 → ▶️ → chọn Bin
  • Nhập 11100011 → =
  • ▶️ → chọn Dec → hiển thị 227
📌 4. Bù 2 (8-bit) của số 7
  • MODE → 4 → chọn Bin
  • Nhập 7 → = → hiển thị 111 (viết đủ 8 bit: 00000111)
  • NOT → hiển thị 11111000
  • + 1 = → 11111001
📌 5. Tính Timer với thạch anh 12MHz (50ms)
  • Công thức: Giá trị nạp = 65536 - (delay_µs)
  • 50ms = 50000µs → 65536 - 50000 = 15536
  • 65536 - 50000 = → 15536
  • ▶️ → chọn Hex → 3CB0 → TH0 = 3C, TL0 = B0
📌 6. Tính 1 chu kỳ máy (11.0592MHz)
  • Công thức: 1 chu kỳ = 12 / tần số thạch anh (MHz)
  • 12 ÷ 11.0592 =
  • Kết quả: 1.085069... ≈ 1.085µs

💡 Mẹo: Nhấn ▶️ để chuyển đổi Dec↔Hex↔Bin
